# Amazon Clone (HTML & CSS)

A clean and responsive Amazon homepage clone built using **pure HTML and CSS**.  
This project focuses on layout structuring, responsive design, and pixel-level attention to detail, without relying on JavaScript or external frameworks.

> This project is part of my frontend development practice and showcases my ability to translate real-world UI designs into clean, maintainable code.

---

## Project Overview

The Amazon Clone replicates the visual structure and layout of the Amazon homepage, including:
- Header and navigation bar
- Search section
- Product showcase sections
- Footer layout

The goal of this project is **not functionality**, but **design accuracy, structure, and responsiveness**.

---

##  Technologies Used

- **HTML5** – Semantic structure and content layout  
- **CSS3** – Styling, Flexbox, positioning, and responsive design  

No JavaScript, frameworks, or libraries were used.

---

## Features

- Responsive layout (desktop-focused)
- Clean and readable HTML structure
- Organized CSS for easy understanding
- Real-world UI cloning practice
